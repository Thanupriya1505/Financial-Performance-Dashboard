# Financial Performance Dashboard

## Project Overview

The Financial Performance Dashboard provides a comprehensive analysis of sales, profit, discounts, and business performance. The dashboard helps stakeholders monitor key financial metrics and make data-driven decisions.

## Tools Used

* SQL
* Power BI
* Tableau
* Excel

## Key Metrics

* Total Sales: 117.83M
* Total Profit: 16.89M
* Units Sold: 1.13M
* Countries Analyzed: 5
* Products Analyzed: 6

## Dashboard Features

* Sales Performance Analysis
* Profitability Analysis
* Product Performance Evaluation
* Country-wise Sales Insights
* Monthly Sales Trend Analysis
* Interactive KPI Dashboard

## Key Insights

* Total Sales reached 117.83M, indicating strong business revenue.
* Total Profit amounted to 16.89M, reflecting healthy profitability.
* Over 1.13M units were sold across multiple markets.
* Operations spanned 5 countries and 6 products, demonstrating a diversified business portfolio.
* Monthly sales trends helped identify growth opportunities and seasonal patterns.

## Benefits

* Supports strategic business decision-making.
* Identifies top-performing products and regions.
* Enables profitability and sales trend analysis.
* Provides a clear overview of financial performance through interactive visualizations.

## Conclusion

This Financial Performance Dashboard transforms raw financial data into actionable insights through data visualization and analytics. It helps businesses monitor performance, improve profitability, and support informed decision-making.
